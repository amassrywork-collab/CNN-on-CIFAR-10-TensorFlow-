# CNN on CIFAR-10 (TensorFlow)

Simple CNN implementation for image classification using CIFAR-10.

## Environment
- Windows
- Python 3.10
- TensorFlow 2.12.1
- NumPy 1.24.3
- Matplotlib 3.7.3

## Run
Open the notebook and run cells top to bottom.

⚠️ Different versions may cause compatibility issues on Windows.
